Middle Years Programme 4

First Term.

Module: Introduction to Scratch Programming.

Duration: 60 minutes.

Objectives:

  1. Introduce students to basic programming and prototyping skills using graphical interface programming.
  2. Ignite curiosity in students through graphical interface programming and fun projects.
  3. Broaden students’ understanding of digital literacy.
  4. Introduce students to the visual programming environment.
  5. Introduce students to programming with multimedia.
  6. Story-telling | music-making | game-making.
  7. Intro to programming concepts such as objects and attributes, sequence, repetition, conditions, and events.
  8. Inculcate a culture of 21st-century skills - communication, creativity, critical thinking, and collaboration.
  9. Increase interest in science-related subjects.

Outcomes:

  1. Ability to write simple programs using graphical interface programming.
  2. Utilisation of science-related concepts such as addiction, subtraction, division and multiplication to write commands.
  3. Proficiency in basic digital literacy.
  4. Proficiency in visual programming.
  5. Creation of simple stories using programming elements such as sequence and repetition.
  6. Appreciation and use of 21st-century skills such as communication, creativity, critical thinking, and collaboration during workshops and in class
  7. Growing interest in science-related subjects such as mathematics and science.
Workshop Topic Activity Number Of Sessions
1. Introduction to basic scratch software What is scratch programming? Why do we learn scratch programming? Installation and launching. Discussion on the interface. 1
2. Scratch programming environment (Section 1) What is a Sprite? What is a backdrop? Understanding the Scratch Libraries. Backdrop libraries. 1
3. Scratch programming environment (Section 2) Scratch main blocks: Script (a set of logic blocks). Costumes (alternate appearances of a sprite). Sounds (these blocks play specified sound events). 2.
4. Scratch programming environment (Section 3) Scratch quick tools (grow, shrink, duplicate, delete). Scratch sub-blocks of the script logic blocks. 2.
5. Mid-Term Assessment (group projects/written). 2.
6. Creation of stories Drag and drop from the sprite library. How to drag drop from the sprite library. Types of sprites. Manipulation of sprites. Adding a backdrop. Creating scratch stories. 2.
7. Costume block What is a costume? Using the costume tools. Using the colour shapes. Sprite manipulation. Computer virtual tools and skills. 3.
8. Combination of knowledge Creation of complex stories. Using combined tools and features. External libraries of backdrops and sprites. Quick research methods for project improvements. 2.
9. End of Term Assessments (presentation/written). 2.

Second Term.

Module: Intermediate Level Scratch Programming.

Duration: 60 minutes.

Objectives:

  1. Introduce students to intermediate-level programming and prototyping skills using scratch.
  2. Introduce students to complex storytelling using advanced programming blocks.
  3. Inculcate a culture of researching and doing presentations.
  4. Broaden students’ understanding of digital literacy.
  5. Inncrease students' skills with computers
  6. Increase student interest in programming
  7. Student achievement on a fun project
  8. Learn Cartesian coordinates, distance computations, etc.
  9. Intro to programming concepts such as objects and attributes, sequence, repetition, conditions and events.
  10. Inculcate a culture of 21st-century skills - communication, creativity, critical thinking, and collaboration.
  11. Increase interest in science-related subjects.

Outcomes:

  1. Ability to write long programs using graphical interface programming.
  2. Utilisation of science-related concepts such as addiction, subtraction, division and multiplication to write commands.
  3. Proficiency in basic digital literacy.
  4. Increased proficiency in graphical interface programming.
  5. Creation of simple stories using programming elements such as sequence and repetition.
  6. Visualisation and creation of stories using programming elements such as sequence, repetition, conditions and events.
  7. Growing interest in science-related subjects such as mathematics and science.
Workshop Topic Activity Number Of Sessions
1. Programming blocks. Blocks and their functions. Intermediate pointing and keying techniques. Turning sprites activity. 1
2. Basic animation. Understanding the motion blocks. How to move and turn sprites. Moving sprite using x-y coordinates. Flying plane hack. 2
3. Creative digital media. Sprite communication logic. Discussion on story creation. Communication between sprites hack. 2.
4. Introductions to events. Introducing event timing. Understanding key press events. How to broadcast. How to hide and show sprites. Events activity (Shoot game). 2.
5. Introduction to scratch sensing blocks. How to broadcast to multiple objects. What is Sensing? How is it done in scratch? Discussion: Programming logic and ideas. Hack: Help the cat escape. 1.
6. Mid-Term Assessment (group projects/written). 2.
7. Object control using block programs. Moving in positive and negative steps. Controlling movement with arrow keys. Understanding how to apply user input. Direction and angles activity hack 2.
8. Basic understanding of basic coordinates. What are coordinates in reference to the Scratch stage size? Understanding pixels. Movement along the x axis activity. Movement along the y axis activity. 1.
9. Basic Angles What is an angle in science, engineering and mathematics? The circle described in degrees and wall corners. (Discussion and practical measurements). Repeated loops and the number of iterations to make in a complete circle.(activity) Simple polygons hack. 2.
10. Introduction to Logical blocks Understanding Variables and Data. What are incrementing and decrementing variables? Examples. Scoreboard hack. 1.
11. Introduction to scratch nested programs. What is a nested program? The design process of long programs. Putting together small program logic to form a program tree. (Discussion and activity). Video. 1.
12. End of Term Assessments (presentation/written). 2.

Third Term.

Module: Introductions to Basic Robotics.

Learning tool:Lego EV3 Mindstorms

Duration: 60 minutes.

Workshop Topic Activity Number Of Sessions
1. Introductions to Lego EV3 robotics.. Understand what a robot is? Review of Lego robots. (Video car robot). Discussing Lego EV3 activities and expectations. Learn and understand applications of robotics. Discussions ,Q/A. 1
2 Lego EV3 Hardware. Identifying Lego EV3 hardware. Understanding sensors. Understanding actuators. Understanding building blocks and materials. Orienting teams and their term plans. Building and assembly activity. 3
3 Intro to EV3 software. Understanding Lego EV3 software interface. What is an IDE? Discussions on various programming methods. Proficiency in drag and drop graphical programing. 1.
4. Mid-Term. Assessments (group projects/written). 2
5. Introductions to Basic EV3 programming. Knowing Basic programming blocks. How to successfully connect Lego EV3 interfacing to a computer. Sample of a program. Programming blocks. My first Lego EV3 program. Move straight hack. 2
6. Motor system programming Learn how to manoeuvre and control the robot in real time. Move and turn hack. 1
7. Touch sensor programming. Introductions to sensing. Learn how to program a touch sensor to respond, or for a task. Learning simple responses. Programming the robot to move until touch. 1
8. Gyroscope and movements programming. What is a gyroscope and its applications? Controlling using angles (Simple Calculation). Program the robot to turn for angle hack activity. 1
9. Ultra sonic sensor Programming Understanding proximity sensing. Application examples(Video demo) Programming the robot to move until near hack activity. 1
10. Combined Parts programming Introductions to repetitive controls. Video demo of multitasking robots and industrial automation for production. Understand and draw basic programing blocks to demonstrate flow of programs. Learn the basic loops hack activity. Program the robot to do multiple tasks. Learn the basics of switches in programming. The Line follower hack activity. 1.
11. End of Term Assessments (presentation/written). 2.
